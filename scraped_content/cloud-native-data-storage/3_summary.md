Modern applications require modern data solutions, especially when catering to audiences on a global scale. Applications like social media or e-commerce platforms rely on non-related data and NoSQL solutions, which makes services like Azure Cosmos DB indispensable.

On this course, we've explored global databases from different perspectives. Our main focus has been on Azure Cosmos DB, the go-to NoSQL solution for Microsoft Azure developers. While there are many NoSQL database services available, Cosmos DB is among the most competent and reliable.

All right, it’s time to review the course contents and take a quick recap of what we’ve learned. Open the following boxes one by one and refresh your memory.
