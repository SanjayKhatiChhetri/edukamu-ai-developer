### Unit 2: Relational and Non-relational Data
</edukamu-collapse-hidden-title>

In the second unit, we explored two different kinds of data: relational and non-relational. We considered many perspectives and tools available in Microsoft Azure before getting to know Azure Cosmos DB.

Consider the following questions. Can you find answers easily? You don’t need to return your answers.

* How would you describe relational and non-relational data?
* Which tools does Microsoft Azure offer for handling databases?
* What are the core functions of Azure Cosmos DB?

</edukamu-collapse>

### Unit 3: Data Analytics in Azure
</edukamu-collapse-hidden-title>

Just having data is not enough, which is why we spend an entire unit focusing on data analytics and visualization. We considered both large-scale and real-time analytics and their most prominent use cases without forgetting Microsoft Power BI, a powerful tool for making clear and effective data visualizations.

Consider the following questions. Can you find answers easily? You don’t need to return your answers.

* What is the difference between large-scale and real-time analytics?
* What aspects should businesses consider when choosing data analytics methods?
* Why is data visualization so important for companies relying on their data?
