Developing modern solutions is a process that involves phases each more challenging, and exciting, as the one before. Microsoft Azure is the world's leading development platform, and you, after completing this course, already have what it takes to start planning your own projects!

On this course, we've taken an in-depth look at solution development with Microsoft Azure. From building web apps to implementing data solutions, we've explored the core functions offered by the platform. Hopefully you've enjoyed the journey!

It’s time to review the course contents and take a quick recap of what we’ve learned. Open the following boxes one by one and refresh your memory.

### Unit 1: Foundations of Azure Development
</edukamu-collapse-hidden-title>

We hit it off by exploring the core principles of Azure development. Quickly we made our way to App Service and Web Apps, while also exploring the opportunities offered by Azure Functions.

Consider the following questions. Can you find answers easily? You don’t need to return your answers.

- What are so-called web apps? How do they differ from other types of applications?
- What are Azure Functions? Give a few examples.
- What kind of solutions can be developed with Azure App Service?

</edukamu-collapse>

<edukamu-collapse isCollapsed="true" title-level="4">
<edukamu-collapse-hidden-title collapseType="aineisto">
### Unit 2: Implementing Data Solutions
</edukamu-collapse-hidden-title>

The second unit was all about data. We started by exploring unstructured data solutions and capabilities for ensuring data availability. From there, we made our way to containerized applications before getting to know Container Apps, the main services for developing such solutions.

Consider the following questions. Can you find answers easily? You don’t need to return your answers.

- What is unstructured data and how can it be handled with Azure?
- How would you describe containerized app solutions?
- In what ways can Azure developers ensure data availability?

</edukamu-collapse>

<edukamu-collapse isCollapsed="true" title-level="4">
<edukamu-collapse-hidden-title collapseType="aineisto">
### Unit 3: Advanced Concepts and Best Practices
</edukamu-collapse-hidden-title>

In the third unit, we focused on a few advanced capabilities offered by Microsoft Azure. We started by reviewing app security and API management, before exploring what else the platform has to offer in terms of event-based and message-based solutions.

Consider the following questions. Can you find answers easily? You don’t need to return your answers.

- What are the main functions of Azure App Configuration?
- Why is API management important when developing solutions?
- What are the differences between event-based and message-based solutions?

</edukamu-collapse>
